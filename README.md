# CybersecurityAwarenessbot
# (Part 1)

## Project Description
Console-based Cybersecurity Awareness Bot developed for PROG6221 - Programming 2A .

**Features implemented:**
- Voice greeting (WAV file)
- Professional ASCII art logo
- Personalised user interaction (name input + validation)
- Keyword-based responses (password, phishing, safe browsing, etc.)
- Input validation and graceful error handling
- Typing effect + coloured console UI
- Clean OOP structure (separate classes)

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio 2022
3. Make sure `Greeting.wav` is in the project root (Copy to Output Directory = Always)
4. Build and run (`F5`)

Or from command line:
```bash
dotnet restore
dotnet build
dotnet run
